# Django aiogram

# Launch
1. Install requirements

   ```bash
     $ pip install -r req.txt
   ```
2. Migrations
  
   ```bash
     $ py manage.py makemigrations
   ```
   ```bash
     $ py manage.py migrate
   ```
3. Run server

   ```bash
      $ py manage.py runserver
    ```
4. Run bot
  ```bash
      $ py manage.py bot_run
    ```
