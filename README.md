# Django aiogram

# Launch
1. Install requirements

   ```bash
     $ pip install -r req.txt
   ```
2. Migrations
  
   ```bash
     $ python manage.py makemigrations
   ```
   ```bash
     $ python manage.py migrate
   ```
3. Run server

   ```bash
      $ python manage.py runserver
    ```
4. Run bot
   ```bash
      $ python manage.py bot_run
    ```
